# 🛍️ Fashion Shopping Website

A responsive fashion e-commerce landing page built using **HTML5, CSS3 and JS**.

This project was created as part of my front-end development practice while restarting my career after a break. The focus was on clean UI design, responsive layout, and reusable styling structure.

---

## 🚀 Live Demo

🔗 [View Live Website] https://prathibhasp.github.io/Shopping-website/

---

## 🛠️ Built With

* HTML5
* CSS3
* JS
* Flexbox
* CSS Variables
* Responsive Media Queries

---

## ✨ Features

* Responsive layout (Desktop, Tablet, Mobile)
* Modern UI design
* Hover effects and smooth transitions
* Product cards with consistent sizing
* Text truncation for long titles
* Reusable button styles
* Organized folder structure

---

## 📱 Responsive Breakpoints

* 992px – Tablet View
* 768px – Mobile Landscape
* 480px – Mobile Portrait
